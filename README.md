# 纯ALOHA仿真

## environment

- python3

## dependence

- random
- numpy
- matplotlib

## Usage

    python3
    from aloha import aloha
    aloha()    # 不进行随机平均

    # or you can:
    aloha(10)  # 进行随机平均，指定次数为10
